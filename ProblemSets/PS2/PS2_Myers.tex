\section{Main Tools of a Data Scientist}

There are many tools a data scientist uses but the main ones are: 
\begin{itemize}
    \item Measurement
    \item Statistical programming languages
        \begin{itemize}
            \item R
            \item Python
            \item Julia
            \item Stata
            \item SAS
            \item SPSS
            \item Matlab
            \item JavaScript
        \end{itemize}
    \item Visualization tools 
        \begin{itemize}
            \item ggplot2 (R)
            \item matplotlib (Python)
            \item Plots.jl (Julia)
            \item Tableau
        \end{itemize}
    \item Big Data management software
        \begin{itemize}
            \item RDDs
            \item SQL
        \end{itemize}
    \item Data collection tools
\end{itemize}
